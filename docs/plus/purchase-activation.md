# Purchase & activation

This documentation page applies to **upstream Koel**. In this fork, **Koel Plus licensing and Lemon Squeezy purchase/activation are not available**. Runtime behavior does not depend on a license key or external payment API.

For upstream documentation, see the [Koel repository](https://github.com/koel/koel).
